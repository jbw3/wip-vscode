# wip-syntax

Syntax highlighting for the WIP programming language.

Note that when cloning this extension into the VSCode extensions directory, the directory name must be the extension name and version number.
For example:

```
git clone git@github.com:jbw3/wip-vscode.git wip-syntax-0.0.1
```
